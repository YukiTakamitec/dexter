# Macro Economic Analysis

Analyze current macroeconomic conditions:

1. **Key Indicators**
   - GDP growth rate (US, Japan)
   - Inflation (CPI, PCE)
   - Employment data (NFP, unemployment rate)
   - PMI (manufacturing, services)

2. **Central Bank Policy**
   - Fed: current rate, dot plot, QT status
   - BOJ: YCC status, rate normalization path
   - Forward guidance and market expectations

3. **Market Regime**
   - Yield curve shape (2y-10y spread)
   - Credit spreads (IG, HY)
   - VIX level and term structure
   - USD/JPY and major FX pairs

4. **Sector Rotation**
   - Current economic cycle phase
   - Leading vs lagging sectors
   - Relative strength analysis

5. **Japan-Specific**
   - BOJ policy normalization impact
   - Yen trajectory and corporate earnings translation
   - Governance reform progress (TSE PBR initiative)
   - Cross-shareholding unwinding trends

Output: Macro outlook summary + sector recommendations + risk factors
