# Technical Analysis

Analyze the stock using the following technical indicators:

1. **Trend Analysis**
   - 50-day and 200-day moving averages
   - Golden cross / death cross signals
   - Price relative to MAs

2. **Momentum Indicators**
   - RSI (14-day): overbought >70, oversold <30
   - MACD (12, 26, 9): signal line crossovers
   - Stochastic oscillator

3. **Volume Analysis**
   - Volume trend (increasing/decreasing)
   - On-Balance Volume (OBV)
   - Volume relative to 20-day average

4. **Support/Resistance**
   - Key price levels from recent highs/lows
   - Fibonacci retracement levels
   - Round number psychology levels

5. **Chart Patterns**
   - Head and shoulders / inverse
   - Double top/bottom
   - Trend channels

Output a concise technical outlook with:
- Current trend direction (bullish/bearish/neutral)
- Key levels to watch
- Signal summary table
